# Инструкция по изображениям

## Структура папок для изображений

```
assets/
  img/
    portfolio/     - изображения портфолио
    services/      - изображения услуг
    clients/       - логотипы клиентов
    icons/         - иконки
```

## Текущие изображения

Сейчас используются бесплатные изображения из Unsplash с прямыми ссылками. Они загружаются автоматически и не требуют скачивания.

## Если хотите использовать свои изображения

1. Поместите изображения в соответствующие папки
2. Обновите пути в `index.html` с `https://images.unsplash.com/...` на локальные пути, например:
   - `/assets/img/portfolio/facade1.jpg`
   - `/assets/img/services/laser-scanning.jpg`
   - и т.д.

## Рекомендуемые размеры изображений

- **Логотип**: 200x60px
- **Категории услуг**: 400x200px
- **Портфолио**: 400x300px
- **Логотипы клиентов**: 200x100px
- **OG изображение**: 1200x630px

## Бесплатные источники изображений

- Unsplash.com - бесплатные фото высокого качества
- Pexels.com - бесплатные стоковые фото
- Pixabay.com - бесплатные изображения

## Тематика изображений для замены

- Геодезические инструменты (тахеометры, нивелиры)
- Строительные площадки
- Архитектурные здания
- Лазерное сканирование
- Топографические карты
