# Руководство по добавлению реальных проектов на карту

## Способ 1: Редактирование JSON файла (Рекомендуется)

### Шаг 1: Откройте файл `assets/data/projects.json`

### Шаг 2: Добавьте новый проект в массив

Пример структуры проекта:

```json
{
  "id": 6,
  "title": "Название вашего проекта",
  "location": "Город, где выполнен проект",
  "coordinates": [59.934543, 30.301235],
  "year": 2024,
  "type": "laser",
  "region": "spb",
  "area": "5 000 м²",
  "date": "Январь 2024",
  "challenge": "Описание задачи проекта",
  "solution": "Как была решена задача",
  "results": [
    "Результат 1",
    "Результат 2",
    "Результат 3"
  ],
  "clientLogo": "URL логотипа клиента (опционально)",
  "photos": [
    "URL фото 1",
    "URL фото 2"
  ],
  "caseLink": "/cases/project-name"
}
```

### Параметры проекта:

- **id** - уникальный номер проекта (число)
- **title** - название проекта (строка)
- **location** - город/адрес (строка)
- **coordinates** - координаты [широта, долгота] (массив из 2 чисел)
- **year** - год выполнения (число: 2021, 2022, 2023, 2024)
- **type** - тип работ:
  - `"facade"` - Фасадная съемка
  - `"laser"` - Лазерное сканирование
  - `"construction"` - Строительная геодезия
- **region** - регион:
  - `"spb"` - Санкт-Петербург
  - `"moscow"` - Москва
  - `"lo"` - Ленинградская область
  - `"mo"` - Московская область
  - `"other"` - Другие регионы
- **area** - площадь объекта (строка, например "5 000 м²" или "10 га")
- **date** - дата выполнения (строка, например "Январь 2024")
- **challenge** - описание задачи (строка)
- **solution** - описание решения (строка)
- **results** - массив результатов (массив строк)
- **clientLogo** - URL логотипа клиента (строка, можно оставить пустым "")
- **photos** - массив URL фотографий (массив строк)
- **caseLink** - ссылка на кейс (строка, например "/cases/project-name")

## Способ 2: Как найти координаты объекта

### Вариант А: Через Яндекс.Карты

1. Откройте https://yandex.ru/maps
2. Найдите адрес вашего объекта
3. Кликните правой кнопкой мыши на точку
4. Выберите "Что здесь?"
5. Скопируйте координаты (они будут в формате: 59.934543, 30.301235)
6. Используйте их в массиве: `[59.934543, 30.301235]`

### Вариант Б: Через Google Maps

1. Откройте https://www.google.com/maps
2. Найдите адрес
3. Кликните правой кнопкой на точку
4. Вы увидите координаты внизу
5. Скопируйте их (формат: 59.934543, 30.301235)
6. Используйте в массиве: `[59.934543, 30.301235]`

## После добавления проекта:

1. Сохраните файл `projects.json`
2. Обновите страницу в браузере (F5)
3. Проект автоматически появится на карте!

## Массовое добавление проектов:

Если у вас много проектов, можно:
1. Создать Excel/CSV файл с данными
2. Конвертировать его в JSON формат
3. Заменить содержимое `projects.json`

## Проверка координат:

После добавления проекта проверьте:
- Правильно ли отображается маркер на карте
- Соответствует ли локация указанному адресу
- Не перекрываются ли маркеры (если проекты рядом, они будут кластеризованы)

## Советы:

- Используйте уникальные ID для каждого проекта
- Проверяйте координаты перед добавлением
- Добавляйте реальные фотографии проектов
- Заполняйте все поля для лучшего отображения в попапе
