## Вариант 2: Telegram Bot для получения заявок

### Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя бота (например: "АМНИС Геодезия")
4. Введите username бота (например: `amnis_geo_bot`)
5. Скопируйте токен бота (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Получение Chat ID

#### Для личных сообщений:
1. Напишите вашему боту любое сообщение
2. Откройте в браузере: `https://api.telegram.org/botВАШ_ТОКЕН/getUpdates`
3. Найдите `"chat":{"id":123456789}` - это ваш Chat ID

#### Для группы/канала:
1. Добавьте бота в группу/канал как администратора
2. Отправьте сообщение в группу/канал
3. Откройте: `https://api.telegram.org/botВАШ_ТОКЕН/getUpdates`
4. Найдите `"chat":{"id":-1001234567890}` - это Chat ID группы/канала

### Шаг 3: Настройка на сайте

Откройте файл `script.js` и найдите строки:

```javascript
const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN';
const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID';
```

Замените на ваши значения:

```javascript
const TELEGRAM_BOT_TOKEN = '123456789:ABCdefGHIjklMNOpqrsTUVwxyz';
const TELEGRAM_CHAT_ID = '123456789';
```

### Шаг 4: Проверка работы

1. Заполните форму на сайте
2. Отправьте заявку
3. Проверьте Telegram - должно прийти сообщение с данными заявки

## Безопасность:

⚠️ **ВАЖНО:** Не публикуйте токен бота в публичном репозитории!

Для продакшена лучше:
1. Использовать переменные окружения
2. Отправлять данные через ваш сервер (backend)
3. Хранить токен в безопасном месте

## Альтернативный вариант: Webhook

Для более безопасной работы можно настроить webhook:

1. Создайте endpoint на вашем сервере
2. Настройте webhook: `https://api.telegram.org/botВАШ_ТОКЕН/setWebhook?url=https://ваш-сайт.ru/api/telegram`
3. Обрабатывайте заявки на сервере и отправляйте в Telegram
